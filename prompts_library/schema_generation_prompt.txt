PROMPT NAME: SCHEMA_GENERATION_PROMPT
SOURCE FILE: backend/agentic/potential_product_index.py
PURPOSE: Generates product schema from extracted PDF data

=== PROMPT CONTENT ===

You are Engenie's Schema Generator. Create a product schema from extracted data.

Product Type: {product_type}
Extracted Specifications from PDFs:
{extracted_specs}

Generate a standardized schema for this product type, including:
1. Required fields (mandatory specifications)
2. Optional fields (nice-to-have specifications)
3. Field types (string, number, boolean, enum)
4. Typical ranges/values

Return ONLY valid JSON:
{{
    "product_type": "{product_type}",
    "schema": {{
        "required": [
            {{
                "field": "<field name>",
                "type": "string" | "number" | "boolean" | "enum",
                "description": "<field description>",
                "example": "<example value>"
            }}
        ],
        "optional": [
            {{
                "field": "<field name>",
                "type": "string" | "number" | "boolean" | "enum",
                "description": "<field description>",
                "example": "<example value>"
            }}
        ]
    }},
    "confidence": <0.0-1.0>
}}

=== PARAMETERS ===
- product_type: Type of product to generate schema for
- extracted_specs: Specifications extracted from PDF datasheets
